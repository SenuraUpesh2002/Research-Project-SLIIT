# FUELWATCH: AI-Powered Smart Surveillance & Predictive Analysis Platform

FuelWatch is a comprehensive station management platform that combines real-time attendance tracking with AI-driven predictive insights.

## Technical Architecture

The platform uses a modular architecture:
1. **Frontend**: React (Vite)
2. **Backend**: Node.js (Express)
3. **AI Service**: Python (Flask)
4. **Database**: MySQL
